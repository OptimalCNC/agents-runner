# Agents Runner

Small web UI for running Codex SDK batches in parallel across git worktrees.
Each batch contains multiple runs, each executing in its own isolated worktree.

For local development setup, commands, and workflow extension notes, see [DEVELOPMENT.md](DEVELOPMENT.md).


## Development Stage

Always use the best approach, do NOT write codes for compatibility, and do NOT write migration codes.
